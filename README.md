# Internship Projects – Data Analytics Dashboards

# Overview
This repository showcases two data analytics projects completed during my internship:
1. Banking Data Dashboard (Credit & Debit datasets) – focused on fraud/risk detection and transaction method analysis.
2. Zomato Dashboard – focused on restaurant and customer behavior analytics.

 Both projects were implemented in Excel, Power BI, Tableau, and SQL, demonstrating end‑to‑end analytics skills across multiple platforms.


# Banking Data Dashboard (Credit & Debit)
- Cleaned and validated banking transaction datasets (credit and debit) for accuracy and consistency.
- Defined KPIs for fraud/risk detection, transaction method analysis, and customer usage patterns.
- Built dashboards in Excel, Power BI, Tableau, and SQL with drill‑downs and filters to highlight suspicious transactions and credit vs. debit trends.
- Documented KPI logic and presented findings with polished visualizations for executives and non‑technical stakeholders.


# Zomato Dashboard
- Cleaned and organized restaurant and customer datasets for analysis.
- Defined KPIs around customer engagement, order frequency, cuisine preferences, and restaurant performance.
- Designed dashboards in Excel, Power BI, Tableau, and SQL to highlight trends in ratings, delivery patterns, and customer behavior.
- Delivered insights with clear documentation and recruiter‑ready layouts.

# Project Impact
- Enabled fraud/risk monitoring and simplified credit vs. debit analysis.
- Revealed customer trends and restaurant performance insights.
